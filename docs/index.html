# Section 1


Definition of symbol $a$ in prose:  $a$ is the slope .

If the symbol also shows up in an equation which is not written by iheartla, it will also be highlighted.
$$a + b = 0$$


The following block shows how to write an iheartla equation
``` iheartla
d = a - c
where
a ∈ ℝ
b ∈ ℝ
```

Symbols can be defined by inline iheartla block as ❤ c = ab  ❤ .

# Section 2
You can define the same symbol in different context:  $a$ is a matrix now .

``` iheartla
d = a
where
a ∈ ℝ^(4 × 4)
```
